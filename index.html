재난관리 4단계 표준 프레임워크

「재난안전법」 제25조의4 ~ 제66조의4 기반
PHASE 01
예방
PREVENTION
재난발생의 원인을 원천적으로 제거하거나, 발생 시 위험도를 줄이기 위한 예방·경감활동
  • 조직 구성 / 규정·제도 개선·정비
  • 재난위험요소 조사·분석
  • 교육·훈련·홍보
  • 재난취약지역·시설 유지관리·점검
  • 운영연속성 확보 (BCP)
PHASE 02
대비
PREPAREDNESS
실제 재난이 발생한 위기상황에서 적절한 대응을 위해 수행해야 할 제반사항을 사전에 준비
  • 예·경보, 모니터링 시스템 구축
  • 장비·물자·시설·인력 확충
  • 규정·제도 개선·정비
  • 교육·훈련
  • 기능별 재난대응 활동계획 수립·점검
PHASE 03
대응
RESPONSE
재난발생 또는 발생이 임박한 상황에서 국민의 생명·신체·재산을 보호하기 위한 활동
  • 재난상황관리 / 긴급생활안정 지원
  • 긴급통신 / 시설피해 응급복구
  • 교통대책 운영 / 의료·방역
  • 재난현장 환경 정비 / 자원봉사
  • 사회질서 유지 / 수색·구조·구급
PHASE 04
복구
RECOVERY
도로·항만·시설물 등 물리적 피해를 재난발생 이전 상태로 원상복구 또는 회복시키는 활동
  • 피해상황·사고원인 조사
  • 복구계획 수립 / 시설복구
  • 피해주민 지원·구호 / 사상자 치료
  • 특별재난지역 선포
  • 평가 및 사고재발대책 마련

사용 설명서
처음 사용하시는 경우 아래 안내를 먼저 확인해 주세요
01
근거
본 플랫폼은 화성시 빅데이터팀이 「2026년 화성시 안전관리계획」(’26.2. / 화성시 안전건설실)에 근거하여 제작한 참조용 자료입니다.

업무 소관은 안전정책과·재난대응과 등 각 소관부서이며, 본 자료는 안전관리계획 내용을 열람·정리한 보조 도구입니다. 공식 효력이 필요한 사항은 소관부서 및 본문 원문을 확인하시기 바랍니다.

상위 법령: 「재난 및 안전관리 기본법」 제24조(시·도 안전관리계획 수립), 동 시행령 제29조(시·군·구 안전관리계획), 행정안전부고시 제2025-80호(’25.12.26.)
02
메뉴 구성
상단 탭으로 4개 화면 전환이 가능합니다.

① 대시보드 — 4단계 표준 프레임워크 / 단계별 추진절차
② 유형별 매뉴얼 — 57개 재난·사고 유형 상세
③ 비상기구 운영 — 대책본부·13개 협업기능
④ 비상연락처 — 유관기관 연락망
03
검색 방법
상단 검색창에 키워드를 입력하면 유형명·부서명·법령·대책명을 동시에 검색합니다.

단축키 Ctrl + K 사용 시 어디서나 즉시 검색창으로 이동합니다. 유형 카드 클릭 시 상세 모달이 열립니다.
04
편집 모드 사용법
우측 상단 「편집 모드」 버튼 클릭 시 모든 정보를 수정할 수 있습니다.

▸ 수정 — 노란 점선 영역 클릭하여 직접 입력
▸ 추가 — "+ 항목 추가" 버튼으로 유형·대책·법령·연락처 추가
▸ 삭제 — 각 항목 우측 × 버튼
▸ 자동 저장 — 모든 변경사항은 브라우저에 자동 저장됨
▸ 백업·복구 — JSON 파일 내보내기/가져오기로 백업 가능
05
유의사항
본 플랫폼은 안전관리계획의 핵심 정보를 정리한 참조용 자료입니다.

상세 통계·세부 추진계획·정확한 수치는 반드시 안전관리계획 본문 원문을 확인해 주세요. 변동사항 발생 시 안전건설실 안전정책과로 즉시 통보해야 합니다.
06
긴급상황 발생 시
우측 상단 「긴급상황 발생」 버튼 클릭 시 유형별 매뉴얼 검색 화면으로 즉시 이동합니다.
